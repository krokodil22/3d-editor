# Mini 3D Editor — полностью оффлайн (без Three.js и без CDN)

✅ Этот проект НЕ использует внешние библиотеки и НЕ делает запросов в интернет.
Работает:
- на GitHub Pages
- на любом хостинге
- локально (даже через file://, но лучше через http.server)

## Запуск локально
В папке проекта:
python -m http.server 8000
Открыть: http://localhost:8000

## Возможности
- Добавление примитивов: куб/сфера/цилиндр/конус/тор
- Выбор объекта кликом
- Move/Rotate/Scale мышью (без гизмо)
- Привязка к сетке (шаг мм)
- Цвет, имя, размеры W/H/D
- Дублирование, удаление, очистка
- Сохранение/загрузка JSON
- Экспорт STL (ASCII)

Если нужно добавить «гизмо» как в Tinkercad — можно сделать, но это уже отдельный модуль.
